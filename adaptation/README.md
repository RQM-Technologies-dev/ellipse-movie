# Adaptation

This folder contains adaptation documents: treatment, beat sheet, screenplay, and scene files.

## Contents

- `treatment.md` — High-level prose summary of the adapted story
- `beat_sheet.md` — Structural beat-by-beat breakdown of the film
- `screenplay/` — Final screenplay drafts in standard format
- `scenes/` — Per-scene YAML files based on `scene_schema.yaml`

## Instructions

Work through adaptation in this order:
1. Write the treatment first to establish scope and compression decisions.
2. Develop the beat sheet to confirm structure.
3. Generate scene files using `tools/scripts/generate_scene_files.py`.
4. Write screenplay scenes referencing the scene files.
