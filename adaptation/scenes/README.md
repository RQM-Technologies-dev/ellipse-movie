# Scenes

This folder contains one YAML file per scene, generated by `tools/scripts/generate_scene_files.py` and populated manually.

## Naming Convention

Files should be named using the scene ID, e.g. `CH01_SC01.yaml`.

## Instructions

1. Run `python tools/scripts/generate_scene_files.py` to create placeholder scene files.
2. Fill in each scene file using `scene_schema.yaml` as a reference.
3. Run `python tools/scripts/validate_yaml.py` to check for parse errors.
