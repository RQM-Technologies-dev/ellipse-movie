# Screenplay

This folder contains screenplay drafts for the adaptation.

## Naming Convention

Files should be named by draft version, e.g.:
- `draft_01.fountain` or `draft_01.md`
- `draft_02_revised.fountain`

## Format

Use standard screenplay format (Fountain markup recommended for plain-text compatibility).

## Instructions

Write scenes referencing the scene YAML files in `adaptation/scenes/` to ensure all story beats are covered.
