# Manuscript

This folder contains the raw manuscript text for the young adult fantasy novel being adapted.

## Contents

- `full_text.md` — The complete manuscript. Upload your text here before running any scripts.
- `chapters/` — Per-chapter splits generated by `tools/scripts/split_chapters.py`.

## Instructions

1. Place the full manuscript text in `full_text.md`.
2. Run `python tools/scripts/ingest_manuscript.py` to verify the file and get a word/page count.
3. Run `python tools/scripts/split_chapters.py` to split into chapter files.
