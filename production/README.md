# Production

This folder contains all production-facing materials: visual bible, shot lists, storyboards, AI prompts, and render outputs.

## Contents

- `visual_bible/` — Style guide, character reference notes, location reference notes
- `shotlists/` — Per-scene shot lists using `shot_schema.yaml`
- `storyboards/` — Storyboard image frames (generated or hand-drawn)
- `prompts/` — AI image and video prompts using `image_prompt_schema.yaml` and `video_prompt_schema.yaml`
- `render_outputs/` — Generated video clips and rendered assets (excluded from git by default)

## Instructions

Work through production in this order:
1. Complete the visual bible before generating any images.
2. Populate shot lists from scene files.
3. Write image prompts for storyboard generation.
4. Generate storyboard frames and review before proceeding to video.
5. Write video prompts and generate 4–8 second clips per shot.
