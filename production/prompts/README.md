# Prompts

This folder contains AI image and video prompt files for storyboard and clip generation.

## Contents

- `image_prompt_schema.yaml` — Schema for storyboard image prompts
- `video_prompt_schema.yaml` — Schema for AI video clip prompts

## Naming Convention

Prompt files should be named using the shot ID, e.g.:
- `CH01_SC01_SH001_image.yaml`
- `CH01_SC01_SH001_video.yaml`

## Instructions

1. Generate image prompts after completing shot lists.
2. Review generated storyboard frames before writing video prompts.
3. Keep prompts versioned — do not overwrite; increment version numbers.
