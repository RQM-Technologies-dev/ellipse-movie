# Visual Bible

This folder defines the visual look of the film: style, character appearances, and location aesthetics.

## Contents

- `style_guide.md` — Overall visual direction, color palette, tone, and cinematography approach
- `character_reference_notes.md` — Visual descriptions of each character for AI image generation
- `location_reference_notes.md` — Visual descriptions of each location for AI image generation

## Instructions

Complete the visual bible before generating any storyboard images or video clips. These documents serve as the reference for all AI image prompts.
