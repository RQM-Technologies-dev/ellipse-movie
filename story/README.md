# Story

This folder contains the story bible for the adaptation: characters, locations, lore, timeline, and continuity notes.

## Contents

- `characters/` — Character profiles using `character_schema.yaml`
- `locations/` — Location profiles using `location_schema.yaml`
- `lore/` — World-building notes, magic systems, history, rules of the world
- `timeline.md` — Chronological event timeline across the story
- `continuity_notes.md` — Running notes on details that must remain consistent

## Instructions

Populate these files as you read through the manuscript. Use the YAML schemas as templates for each character and location entry.
