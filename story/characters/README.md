# Characters

This folder contains one YAML file per character, based on `character_schema.yaml`.

## Naming Convention

Files should be named after the character ID, e.g. `CHAR_ELARA.yaml`.

## Instructions

1. Copy `character_schema.yaml` and rename it for each character.
2. Fill in all applicable fields.
3. Use `continuity_notes` to track details that must remain consistent across scenes.
