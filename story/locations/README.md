# Locations

This folder contains one YAML file per location, based on `location_schema.yaml`.

## Naming Convention

Files should be named after the location ID, e.g. `LOC_MOONWATER_BRIDGE.yaml`.

## Instructions

1. Copy `location_schema.yaml` and rename it for each location.
2. Fill in all applicable fields.
3. Use `visual_reference_notes` to describe the look for AI image generation.
