# Lore

This folder contains world-building notes derived from the manuscript.

## Suggested Files

- `magic_system.md` — Rules, costs, and limits of any magic in the story
- `history.md` — Key historical events referenced in the story
- `factions.md` — Groups, organizations, or political entities
- `creatures.md` — Non-human beings, monsters, or fantastical animals
- `artifacts.md` — Significant objects with narrative or magical importance

## Instructions

Create one file per topic as needed. Reference specific chapters or scenes where lore is established so continuity can be tracked.
