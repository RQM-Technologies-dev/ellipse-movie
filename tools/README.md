# Tools

This folder contains Python scripts for manuscript ingestion, chapter splitting, scene file generation, and YAML validation.

## Contents

- `scripts/` — Python scripts for pipeline automation

## Requirements

Install dependencies before running any scripts:

```bash
pip install -r requirements.txt
```

## Instructions

Run scripts from the repository root:

```bash
python tools/scripts/ingest_manuscript.py
python tools/scripts/split_chapters.py
python tools/scripts/generate_scene_files.py
python tools/scripts/validate_yaml.py
```
